# Multi-Agent Stock Market Simulator 📈

This project is an end-to-end Agentic AI system designed for the SEN4018 Semester-Long Project. It simulates a dynamic stock market environment where autonomous agents collaborate to make data-driven trading decisions. A core feature of this project is the pervasive use of Natural Language Processing (NLP) across all agents to parse, synthesize, and generate financial insights.

## 💡 Project Motivation & Problem Definition
Financial markets are influenced by a complex mix of quantitative data and qualitative information. This project addresses the challenge of combining technical and fundamental analysis by deploying autonomous AI agents that operate without human intervention. By integrating NLP at every stage, the system not only processes numbers but also "reads" the market context and explains its decisions in human-readable language.

**Fields:** Artificial Intelligence, Natural Language Processing (NLP), Multi-Agent Systems, and Financial Engineering.

## 🏗️ System Architecture & NLP Integration Flow
The system is built on a multi-agent architecture where distinct AI agents operate in a continuous loop, heavily utilizing NLP:

### Technical Analysis Agent 📊
- **Function:** Analyzes stock price trends and calculates precise indicators (Moving Averages, RSI, MACD) via Python.
- **NLP Role:** Uses Natural Language Generation (NLG) to convert raw mathematical indicators into contextual textual summaries, and parses written technical analyst reports to extract support/resistance levels.

### Fundamental Analysis (News) Agent 📰
- **Function:** Scans real-time financial news and market updates.
- **NLP Role:** Applies advanced sentiment analysis, Named Entity Recognition (NER), and text summarization to extract bullish/bearish signals from unstructured news articles and press releases.

### Portfolio Manager Agent 💼
- **Function:** Synthesizes outputs from both the Technical and Fundamental agents to make the final "Buy", "Sell", or "Hold" execution command.
- **NLP Role:** Instead of just comparing numerical weights, it reads the textual arguments generated by the sub-agents. It uses NLP to synthesize these arguments into a comprehensive, human-readable "Investment Thesis" explaining the core reasoning behind its autonomous decision.

### LLM-in-the-Loop Evaluation 🔍
A final supervisor prompt reviews the Portfolio Manager's written investment thesis to ensure the logic aligns with standard financial practices and checks for cognitive bias.

## ⚙️ Datasets & Technologies
- **API & Data Sources:** yfinance (Yahoo Finance API) for market data, and NewsAPI/Finnhub for real-time financial articles.
- **Agent Orchestration:** LangChain (managing tool execution, memory, and multi-agent reasoning loops).
- **LLM Engine:** Google Gemini 1.5 Pro via API for robust reasoning, advanced NLP tasks, and large context window processing.
- **Data Processing:** Python (Pandas, NumPy) for numerical arrays, coupled with NLP libraries (NLTK, spaCy, or directly via LLM prompts).
- **User Interface & Storage:** Streamlit for interactive visualizations. Features a live Dashboard and a persistent **History Tab** (backed by a local JSON database) to review and backtest past AI decisions without consuming additional API quotas.
- **Rate Limit Handling:** Built-in delay mechanisms to gracefully handle Google Gemini API rate limits (Free Tier quotas) during multi-agent sequential execution.

## 🚀 Getting Started (Installation)

To run this project locally, follow these steps:

**1. Clone the repository:**
```bash
git clone https://github.com/KULLANICI_ADINIZ/Multi-Agent-Stock-Market-Simulator.git
cd Multi-Agent-Stock-Market-Simulator
```

**2. Create a virtual environment and install dependencies:**
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

**3. Configure Environment Variables:**
Create a `.env` file in the root directory and add your API keys:
```env
GOOGLE_API_KEY="your_google_gemini_api_key_here"
NEWS_API_KEY="your_newsapi_key_here"
```

**4. Run the Application:**
Launch the Streamlit interface:
```bash
streamlit run app.py
```
Open your browser at `http://localhost:8501` to use the simulator.
